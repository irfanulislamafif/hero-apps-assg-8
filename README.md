# GemStore - Hidden Gems App Discovery

## ðŸ“Œ Project Overview
**GemStore** is a curated App Store application designed to help users discover unique, high-utility, and "hidden gem" mobile applications that often fly under the radar. Unlike standard app stores that focus on trending games, GemStore focuses on creativity, human connection, and genuine utility.

This project was built to demonstrate proficiency in **React.js**, **Routing**, **Data Visualization**, and **State Management** (including LocalStorage).

### 1. Forest: Stay Focused
* **Description:** A gamified productivity app that helps users beat phone addiction. Users plant virtual trees that grow while they focus but wither if they leave the app.

### 6. RunPee
